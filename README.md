# WiP ATAK map overlay for JBLM TAs. 
Currently includes entire JBLM boundary, TA 06, TA 11-23\
Additional overlays for some roads/trails in TAs 19-23\


## Installation:
### WinTAK:
Download:\
Click on JBLM_TAs.zip\
Click View raw\
Navigate to your Downloads folder and extract the zip folder

In WinTAK:\
Click on hamburger menu in top left corner\
Hover mouse over Import Manager\
Click on Import Directory\
Click on JBLM_TAs

### ATAK:
Download:\
Click on JBLM_TAs.zip\
Click View raw\
Do not extract zip folder

In ATAK:\
Tap on hamburger menu in top right corner\
Tap Import
Tap Local SD
Tap on Download (If you do not see Download folder tap the back arrow in upper left corner until it does nothing)\
Tap Date twice\
Tap JBLM_TAs.zip\
Tap Move\
If import fails keep repeating steps until it works

## Sources:
iSportsman for boundaries ([jblm.isportman.net](https://jblm.isportsman.net/Open/AreaMap.aspx))\
ArcGIS for road names ([arcgis.com](https://www.arcgis.com/apps/mapviewer/index.html))
